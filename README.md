# React Big Calendar Dashboard

A React-based dashboard that visualizes **date-wise data** using **React Big Calendar** and an **interactive bar chart**.  
Users can select dates from the calendar to view corresponding data in a popup with graphical representation.

---

## Tech Stack

- **React + Vite**
- **Redux Toolkit** – State management
- **React Big Calendar** – Calendar UI
- **Material UI (MUI)** – Modern UI components
- **amCharts 5** – Bar chart visualization
- **date-fns** – Date handling utilities

---

## Features

- Month / Week / Day calendar views
- Highlighted dates with available data
- Click on a date to open a popup
- Bar chart visualization for selected date data
- Clean and modern Material UI design
- Responsive layout for desktop and mobile
